# NITAD Poll Frontend

## Package Installation

_pnpm_ is required to install the package.

```bash
npm i -g pnpm
```

install dependencies

```bash
pnpm i
```

## Development

To run a dev server

```bash
pnpm run dev
```

To build the app for production

```bash
pnpm run build
```

To preview the built app

```bash
pnpm run preview
```

## Tech Stack

svelte + sveltekit + tailwindcss
